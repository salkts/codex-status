# Changelog

## 0.1.0

- Initial public release candidate.
- Adds native macOS menu bar status icon and timer strip for local Codex turns.
- Tracks active turns from local Codex rollout logs and SQLite thread index.
- Shows multiple active timers with overflow handling.
- Adds settings for timer strip visibility, max visible timers, icon style,
  shimmer cadence, and launch at login.
- Adds short completed-turn timer state for main chats.
- Adds build, run, package, and verification scripts.
